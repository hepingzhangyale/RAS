# Data files (download separately)

The genotype and phenotype files are **not stored in this repository**. Download
them from Figshare and place all four files in this `data/` folder before running
`../run_ras_pig.R`:

- Duroc pig genotype and phenotype dataset — Figshare
  [10.6084/m9.figshare.4263317](https://doi.org/10.6084/m9.figshare.4263317)

Expected files:

```
data/GWAS_genotypes.bed
data/GWAS_genotypes.bim
data/GWAS_genotypes.fam
data/gwas_duroc_phenotypes.xlsx
```
